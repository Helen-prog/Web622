Базовые технологии

Технологии для front-end разработки

HTML (HyperText Markup Language)
Отвечает за структуру и содержание веб-страницы.

CSS (Cascading Style Sheets)
Определяет внешний вид элементов (цвета, шрифты, расположение) и обеспечивает адаптивный дизайн с помощью таких технологий, как Flexbox и Grid.

JavaScript
Добавляет интерактивность и динамическое поведение на страницу, обрабатывает события и взаимодействие с сервером.
